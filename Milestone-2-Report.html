Security Posture

  • JWT dual transport — Bearer header or accessToken HttpOnly cookie; 15-minute access tokens, 7-day DB-backed refresh tokens with rotation and reuse detection.
  • Login lockout — failedLoginCount / lockedUntil per user; resets on success.
  • Global guard chain — ThrottlerGuard → JwtAuthGuard → RolesGuard → OwnershipGuard; auth routes additionally throttled (configurable via env).
  • Idempotency — POST /bookings and POST /payments/initiate accept Idempotency-Key header; replay returns identical response, same key with different body → 409.
  • Argon2id for all password and OTP token hashing.
  • No OTP enumeration — forgot-password always returns 200; OTP errors return 422 not 400.
  • Helmet, CORS, compression on all responses; cookie-parser for token cookies.
